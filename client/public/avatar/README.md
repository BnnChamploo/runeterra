# 英雄头像文件夹

## 说明

这个文件夹用于存放英雄联盟英雄的头像图片。

## 文件命名规则

头像文件应该按照英雄的中文名命名，格式为：`英雄中文名.png`

例如：
- `亚托克斯.png` - 暗裔剑魔
- `阿狸.png` - 九尾妖狐
- `阿卡丽.png` - 离群之刺
- `艾希.png` - 寒冰射手
- ... 等等

## 支持的格式

- `.png`（推荐）
- `.jpg` / `.jpeg`
- `.webp`

## 使用方法

1. 将英雄头像图片放入 `client/public/avatar/` 文件夹
2. 文件名使用英雄的中文名（见 `client/src/data/heroes.js`）
3. 系统会自动识别并显示对应的头像

## 示例

```
client/public/avatar/
  ├── 亚托克斯.png
  ├── 阿狸.png
  ├── 阿卡丽.png
  ├── 艾希.png
  └── ...
```

## 注意事项

- 如果某个英雄的头像文件不存在，会显示默认头像占位符（首字母）
- 图片建议尺寸：200x200 像素或更大
- 建议使用透明背景的 PNG 格式
